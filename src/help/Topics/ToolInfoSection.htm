Measurement information for the currently selected tool is displayed in the Tool Information section. The name of the current measurement tool is displayed at the top of the section. Depending on the tool selected, various text fields in this section will be enabled for input and display. The exact meaning of each text field is dependent on the selected measurement tool.
The position text fields (i.e. X1/Y1, X2/Y2, XV/YV) provide a pair of up/down buttons that allow fine adjustment of the tool position.
X1, Y1 - Position 1
X2, Y2 - Position 2
XV, YV - Angle vertex position or circle center position
W, H - Width and height
D - Distance or radius
A - Angle
Ar - Area
Note that the calculation of W, H, D and Ar is always performed in pixel units and the result is converted to the currently selected display units.
